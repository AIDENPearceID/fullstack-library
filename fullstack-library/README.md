# fullstack-library

## المتطلبات
- Node.js >= 18
- MongoDB (محلي أو Atlas)

## تشغيل backend
cd backend
cp .env.example .env   # غيّر القيم
npm install
npm run dev

## تشغيل frontend
cd frontend
npm install
npm run dev

## رفع إلى GitHub
- إنشئ repo جديد
- `git init`, `git add .`, `git commit -m "init"`, `git remote add origin ...`, `git push -u origin main`
